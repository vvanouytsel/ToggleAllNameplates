# ToggleAllNameplates

A World of Warcraft addon that automatically controls whether all nameplates are shown, based on your current zone.

## How it works

When you enter a zone that is enabled in the settings, `nameplateShowAll` is turned on — meaning nameplates are visible above all units at all times. When you leave that zone (or enter one that is not enabled), it is turned off and nameplates only appear when you target or interact with a unit.

## Configuration

Open **Game Menu → Options → AddOns → ToggleAllNameplates** to choose which zone types should have all nameplates shown:

- Open World
- Dungeons
- Raids
- Scenarios
- Battlegrounds
- Arenas

## Slash commands

| Command | Action |
|---|---|
| `/tan` | Manually toggle nameplates on/off |
| `/tan config` | Open the settings panel |
